# Contracts

Payment and chain layer for PerkOS Knowledge.

Initial chains:

- Base — primary x402/EVM rail
- Celo — mobile/global stablecoin EVM rail
- Solana — high-throughput micropayment rail

MVP may not need custom contracts. Use this directory for:

- chain configs
- deployment notes
- x402 settlement adapters
- optional receipt registry
- optional subscription/access pass contracts
- Foundry/Hardhat scripts if EVM contracts are needed
- Solana program/scripts if needed later
